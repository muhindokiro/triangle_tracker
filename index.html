Triangle Tracker

This is a triangle tracking program that helps you check whether the values you have given are to check whether they can form an equilateral, isosceles, scalene or not form a triangle at all. First you input values in the given spaces then check for the response which will be either equilateral, isosceles, scalene or not a triangle at all depending on the values you have given.
